# Практична робота №4
Цей репозиторій cтворений для перегляду виконання практичної роботи №4 з дисципліни "Технології збору та обробки даних", виконане студентом Щур Р.І., групи ТВ-32.

## Мета роботи:
Реалізація байєсівської регресії та її порівняння з класичною лінійною регресією.
## Поставлені завдання:
Реалізуйте байєсівську регресію для моделювання залежностей між погодними умовами та енергоспоживанням. Порівняйте отримані висновки з результатами, отриманими за допомогою класичної лінійної регресії. Побудуйте графіки для візуалізації отриманих залежностей.

## Програмна реалізація:

Для подальшої роботи програми було згенеровано набір тестових даних. Для відтворюваності результатів під час майбутніх запусків програми використовується фіксація стану генератора випадкових чисел за допомогою np.random.seed(42). Набір даних становить лінійна послідовність з 50 значень температури в діапазоні від 10°C до 35°C. Змінна енергоспоживання розраховується на основі теоретичної лінійної моделі з базовим споживанням 10 одиниць та коефіцієнтом чутливості 2.5 до кожного градуса температури. Для наближення моделі до реальних умов експлуатації, до кожного значення додається випадковий шум, розподілений за нормальним законом із нульовим середнім та стандартним відхиленням у 7 одиниць. Згенеровані дані структурується у форматі pandas.DataFrame.
